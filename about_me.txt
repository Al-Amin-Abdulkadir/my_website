Nebula OS (concpt web/app design)
==================================

Nebula OS is designed to make civilian space travel structured, safe, and accessible. While space tourism has long been limited by complexity, high risk, and fragmented systems, Nebula OS brings everything into a single intelligent interface from navigation and gravity control to communication and passenger experience. Noir Brutalism (Concept Magazine design)
==========================================

Noir brutalism is mainly inspired by brutalist architecture and darker visual aesthetics. The focus is on large, heavy structures, strong geometric forms, and spaces that feel massive compared to the people in them. salary prediction Machine Learning prediction system (Machine Learning)
===========================================================================


I aspire to be an ML Engineer one day and i have been working towards  that..i have a project/experiment to prove that

This is a basic salary prediction project. I wouldn't pass it off as a project, but as an experiment done on different models and how they would make predictions and decisions based on the same dataset. I started with a mock dataset, used pandas library to clean, remove duplicates and feel empty brackets if there was any, then proceeded to use train test split to analyze the predictions of the model based on training data and data it has not seen before, followed by an evaluation test to see which model was the best fit for salary prediction. I also used matplotlib to visualize the linear line.

I trained three models : 

1. Linear Regression Model
2. Decision tree model
3. Random forest

The final evaluation was analyzed through metrics including Mean Absolute Error and r2 score which translate to "how wrong the model is on average" and "how the model explains variance". These are the numbers : 

Linear regression :
MAE : 2500.000000000022
r2 score : 0.9921875

Decision Tree : 
MAE : 15000.0
r2 score : 0.84375

Random Forest : 
MAE : MAE: 6800.0
r2 score : 0.955475

from the numbers above, I can state that linear regression performs well. To back up my statement on the winner, linear regression performs because the dataset showcases a linear relationship between experience, age, and salary which means that the salary increased proportionally with experience. It also has the lowest Means Absolute Error which me means lower error rate during prediction 

During further research, i learned that models such as decision trees and random forest are more complex models that are capable of capturing non linear patterns. The dataset was to simple which in turn introduced over fitting, a term whereby the model is to complex and learns everything including noise and unreliable features.
